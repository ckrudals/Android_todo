데이터베이스의 종류는 5가지 로 구분됩니다.

아주 간략하게 설명하자면

**계층형 데이터베이스**

\-> 데이터를 계층 구조로 관리하는 데이터베이스, 조직도나 전체구조도를 상상하면 이미지가 쉽게 잡힌다.

**관계형 데이터베이스**

\-> 2차원 표 형식으로 데이터를 관리하는 데이터베이스, 현재 가장 많이 쓰인다.

**객체지향 데이터베이스와 XML 데이터베이스**

\-> 각각 **객체** 와 **XML** 형식으로 관리하는 데이터베이스, 많이 접할 기회는 없다.

**NoSQL 데이터베이스**

\-> NoSQL 이란, NotOnlySQL이다, 이말은 SQL 말고 다른 것이 더 있다는 의미이다. 관계형 데이터베이스에 있는 기능을 일부 버려 속도를 높이고 있다. 고속으로 데이터를 처리해야하는 웹서비스에 많이 이용된다.
